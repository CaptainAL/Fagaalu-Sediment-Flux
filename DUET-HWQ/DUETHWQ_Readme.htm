How to Use DUET-H/WQ

System Requirements

DUET-H/WQ is distributed as an installable Windows file that requires the “Microsoft.NET 2.0 Framework with Service Pack 2”.  The setup program will check to see if your computer has the required “Microsoft.NET 2.0 Framework with Service Pack 2.”  If the Framework is not installed, the setup program will guide you through the installation process.

 

 

The following is a step-by-step overview of how to use DUET-H/WQ to estimate uncertainty in measured discharge, concentration, and load data. 

 

How to Import a Raw Data File

1.      Click “Import Data File” in the “File” pulldown menu.  The raw data file must be a MS-DOS text file (tab delimited).  For other file format requirements, see the “Import Datafile Format” instructions in the “Help” menu.  The software will insert an uncertainty column next to each data column.  Upon exiting the software, you will be asked to save your work. A DUET-H/WQ file can be created if you select “Yes” in the pop up box. 

 

2.      When you reopen this file, select “Open DUET-H/WQ File” on the “File” pulldown menu.  If you choose “Import File,” unnecessary uncertainty columns will be added to the file.

 

 

How to Calculate Uncertainty

1.      Click on the first cell in the first uncertainty column.  This will activate the pulldown menus needed to calculate uncertainty.

 

2.      In the “Calculate Uncertainty” pulldown menu, select the data type (only discharge, concentration, or load are options) in the adjacent column for which you will calculate the uncertainty.

 

3.      If you choose “Calculate Uncertainty for Discharge,” two options will be available.  Choose “Use default discharge uncertainty” to select a default uncertainty estimate or choose “Calculate discharge uncertainty” to use project specific data, published information, and professional judgment to calculate uncertainty.

 

3.1.   If you choose “Use default discharge uncertainty,” default uncertainty estimates for a range of date quality scenarios will pop up.  Choose an appropriate uncertainty estimate.

 

3.2.   If you choose “Calculate discharge uncertainty,” a lookup table to assist in calculating the uncertainty will pop up.  Select a method in each applicable section to view published uncertainty information.  This will also place an uncertainty value for that method into the appropriate box.  If that uncertainty is not appropriate for your conditions, change the value.

 

3.3.   The uncertainty contributed by Data Processing and Management can be added (if known) to either the default or calculated uncertainty.  For more information on this source of uncertainty, see the “What is DPM Uncertainty?” in the “Help” pulldown menu.  To include Data Processing and Management uncertainty, select “Enter the Data Processing and Management Uncertainty” and enter the uncertainty.

 

3.4.   The software will then calculate the uncertainty for that discharge value.  The uncertainty will include Data Processing and Management uncertainty if a value for that source of uncertainty is entered.

 

4.      If you choose “Calculate Uncertainty for Concentration,” two options will be available.  Choose “Use default concentration uncertainty” to select a default uncertainty estimate or choose “Calculate concentration uncertainty” to use project specific data, published information, and professional judgment to calculate uncertainty.

 

4.1.   If you choose “Use default concentration uncertainty,” default uncertainty estimates for several constituents for both storm and baseflow for a range of data quality scenarios will pop up.  Choose an appropriate uncertainty estimate.

 

4.2.   If you choose “Calculate concentration uncertainty,” then menu options for the three procedural categories (Sample Collection, Preservation and Storage, and Laboratory Analysis) necessary for concentration data will pop up.  The uncertainty in each of these categories should be calculated before proceeding.

 

4.2.1.      For each procedural category, a lookup table will pop up to assist in calculating the uncertainty.  Select a method in each applicable section to view published uncertainty information.  This will also place an uncertainty value for that method into the appropriate box.  If that uncertainty is not appropriate for your conditions, change the value.

 

4.3.   The uncertainty contributed by Data Processing and Management can be added (if known) to either the default or calculated uncertainty.  For more information on this source of uncertainty, see the “What is DPM Uncertainty?” in the “Help” pulldown menu.  To include Data Processing and Management uncertainty, select “Enter the Data Processing and Management Uncertainty” and enter the uncertainty.

 

4.4.   The software will then calculate the uncertainty for that concentration value.  The uncertainty will include Data Processing and Management uncertainty if a value for that source of uncertainty is entered.

 

5.      If you choose “Calculate Uncertainty for Load,” two options will be available.  Choose “Use default load uncertainty” to select a default uncertainty estimate or choose “Calculate load uncertainty” to use project specific data, published information, and professional judgment to calculate uncertainty.

 

5.1.   If you choose “Use default load uncertainty,” default uncertainty estimates for several constituents for a range of data quality scenarios will pop up.  Choose an appropriate uncertainty estimate.

 

5.2.   If you choose “Calculate load uncertainty,” then menu options for the four procedural categories (Discharge, Sample Collection, Preservation and Storage, and Laboratory Analysis) necessary for load data will pop up.  The uncertainty in each of these categories should be calculated before proceeding.

 

5.2.1.      For each procedural category, a lookup table will pop up to assist in calculating the uncertainty.  Select a method in each applicable section to view published uncertainty information.  This will also place an uncertainty value for that method into the appropriate box.  If that uncertainty is not appropriate for your conditions, change the value.

 

5.3.   The uncertainty contributed by Data Processing and Management can be added (if known) to either the default or calculated uncertainty.  For more information on this source of uncertainty, see the “What is DPM Uncertainty?” in the “Help” pulldown menu.  To include Data Processing and Management uncertainty, select “Enter the Data Processing and Management Uncertainty” and enter the uncertainty.

 

5.4.   The software will then calculate the uncertainty for that load value.  The uncertainty will include Data Processing and Management uncertainty if a value for that source of uncertainty is entered.

 

 

How to Apply the Uncertainty Estimate to Other Measured Values in That Column

1.      The last uncertainty calculated will appear in the box in the pulldown menu heading line.  That uncertainty estimate can be applied to the entire column of measured data or can be applied to a range of measured values.  This eliminates the necessity of repeating the previous steps for every measured value.

 

1.1.   To apply the uncertainty estimate to a range measures values, highlight the desired range of cells in the appropriate uncertainty column.  Then select “Apply Value in Box to Selected Cell(s)” in the “Apply Value to Cell(s)” pulldown menu.

 

1.2.   To apply the uncertainty estimate to the entire column, highlight at least one cell in the appropriate uncertainty column.  Then select “Apply Value in Box to Entire Column of Selected Cell in the Apply Value to Cell(s)” pulldown menu.
